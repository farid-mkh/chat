# Socket.IO Chat example clone

1. backend: node.js and socket.io
2. frontend: html, css, js 

## How to run
> First make sure that you have installed node.js.
1. To install packages 
```npm i``` or 
```yarn install```
2. To run the app 
```node app.js```
